#### Dispenser Backend Routes

| Method | Url                                | Description  |
|--------|------------------------------------|--------------|
| GET    | http://127.0.0.1:8000/             | Default      |
| GET    | http://127.0.0.1:8000/status-check | Status check |

Default
```bash
http GET http://127.0.0.1:8000/
```

Status check
```bash
http GET http://127.0.0.1:8000/status-check
```
